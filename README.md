# SegawaBeer

你好，我是 SegawaBeer，一名 AI 从业者，也是一名在企业现场长期打磨出来的 FDE / 咨询顾问 / PM 复合型选手。

我目前在大型互联网公司从事 AI 工程与解决方案相关工作，主要面对真实企业场景，把业务问题拆成可建模、可实现、可落地的 AI 应用和系统方案。简单说，就是在客户现场和产品技术之间来回穿梭，把“这个能不能用 AI 做？”翻译成“应该怎么做、先做哪一步、最后怎么上线”。

在此之前，我曾在头部咨询公司担任企业咨询顾问，参与过数字化转型、数据治理、流程优化、指标体系、AI 规划等项目。咨询经历让我更习惯从业务全局看问题：不只看一个功能能不能做，更看组织、流程、数据、系统和人之间到底卡在哪里。

更早一些，我也做过企业级产品经理和产品负责人，覆盖金融科技、风控数据产品、决策系统和 B 端平台设计。这段经历让我对“产品不是 PPT，系统不是口号”这件事有比较深的体感。

所以我的职业标签大概是：

- **AI Engineer / FDE**：把 AI 能力带进真实业务场景
- **Consultant**：理解企业复杂问题，并把它讲清楚、拆明白
- **PM**：把需求、方案、系统和交付路径串起来
- **Solution Builder**：不只提建议，也关心东西最后能不能跑起来

我关注的方向包括 AI Agent、企业 AI 应用、知识系统、数据治理、业务建模和个人自动化。这里会慢慢整理一些我觉得值得公开的实践：不追求仓库数量很多，但希望每个项目都能说明一个真实问题，以及我是怎么理解、拆解和解决它的。
